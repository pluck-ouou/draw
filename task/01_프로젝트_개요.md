# 송년회 뽑기 게임 - 프로젝트 개요

## 프로젝트명
**Lucky Draw** (럭키 드로우)

## 프로젝트 목적
송년회에서 참가자들이 QR코드를 통해 간편하게 접속하여 실시간으로 뽑기에 참여할 수 있는 웹 기반 게임

## 핵심 컨셉
- 문방구 뽑기판 스타일의 100칸 뽑기 게임
- 로그인 없이 이름만 입력하면 바로 참여 가능
- 실시간 동기화로 중복 뽑기 방지
- 모바일 친화적 UI

## 주요 요구사항

### 1. 접근성
- QR코드 스캔으로 즉시 접속
- 회원가입/로그인 불필요
- 참가자 이름만 입력하면 참여 가능

### 2. 게임 규칙
- 총 100개의 뽑기 칸
- 각 칸에는 경품 또는 꽝이 배정됨
- 한 번 뽑힌 칸은 다른 사람이 뽑을 수 없음
- 1인 1회 참여 제한 (선택적)

### 3. 실시간 동기화
- 누군가 뽑으면 모든 참가자 화면에 즉시 반영
- 새로고침 없이 자동 업데이트
- 동시 클릭 시 충돌 방지

### 4. 관리자 기능
- 경품 설정 및 배치
- 게임 시작/종료 제어
- 결과 현황 모니터링

## 타겟 사용자
- 송년회 참가자 (다수의 동시 접속)
- 관리자 (이벤트 운영자)

## 성공 기준
- 50명 이상 동시 접속 처리
- 중복 뽑기 0건
- 3초 이내 실시간 반영
