### 3.2 칸(Slot) 상태별 디자인
| 상태 | 배경색 | 테두리 | 텍스트 | 효과 |
|------|--------|--------|--------|------|
| 미선택 (사용가능) | 흰색/밝은색 | 골드 | 번호 (진한색) | 호버 시 확대 |
| 뽑힘 | 회색 | 없음 | 뽑은 사람 이름 | 투명도 낮춤 |
| 선택중 (애니메이션) | 골드 | 두꺼운 골드 | 번호 | 펄스 효과 |
| 내가 뽑음 | 강조색 | 두꺼운 테두리 | 내 이름 | 반짝임 |

### 3.3 반응형 그리드
```css
/* 모바일 (기본) */
.grid {
  display: grid;
  grid-template-columns: repeat(5, 1fr);  /* 5x20 */
  gap: 4px;
}

/* 태블릿 */
@media (min-width: 640px) {
  .grid {
    grid-template-columns: repeat(10, 1fr);  /* 10x10 */
    gap: 8px;
  }
}

/* 데스크톱 */
@media (min-width: 1024px) {
  .grid {
    max-width: 800px;
    gap: 12px;
  }
}
```

## 6. 색상 팔레트

### 6.1 메인 컬러
```css
:root {
  /* 프라이머리 - 골드/축제 분위기 */
  --primary-50: #fffbeb;
  --primary-100: #fef3c7;
  --primary-500: #f59e0b;
  --primary-600: #d97706;
  --primary-700: #b45309;

  /* 세컨더리 - 레드/축하 */
  --secondary-500: #ef4444;
  --secondary-600: #dc2626;

  /* 배경 */
  --bg-primary: #1a1a2e;    /* 어두운 배경 */
  --bg-secondary: #16213e;  /* 약간 밝은 배경 */
  --bg-card: #ffffff;       /* 카드 배경 */

  /* 텍스트 */
  --text-primary: #ffffff;
  --text-secondary: #a0aec0;
  --text-dark: #1a202c;

  /* 상태 */
  --success: #10b981;
  --error: #ef4444;
  --warning: #f59e0b;
}
```

### 6.2 다크/라이트 테마
축제 분위기를 위해 **다크 테마** 권장 (골드 색상이 더 돋보임)

## 9. 모바일 UX 고려사항

1. **터치 영역**: 최소 44x44px
2. **폰트 크기**: 최소 16px (확대 방지)
3. **스크롤**: 뽑기판 전체가 화면에 보이도록
4. **진동 피드백**: 뽑기 시 햅틱 피드백 (선택)
5. **방향 고정**: 세로 모드 권장

## 10. 접근성

- 키보드 네비게이션 지원
- 색맹 사용자를 위한 아이콘/패턴 병용
- 스크린 리더 지원 (aria-label)
- 충분한 색상 대비 (WCAG AA 기준)
